# Sorting Hill


## Работа с зависимостями
В качестве менеджера зависимостей используется `uv`.
Установить его можно по инструкции - https://docs.astral.sh/uv/getting-started/installation/

## Команды запуска:
Для запуска проекта (и не только) используется `Makefile`.

Для того, чтобы синхронизировать зависимости:
```bash
make sync
```

Для того, чтобы запустить тесты:
```bash
make test
```

Для запуска проекта:
```bash
make run
```

Для проверки кода линтером:
```bash
make linter
```
